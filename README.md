# Blackjack Web App

## Overview
A browser-based Blackjack game built with vanilla JavaScript to demonstrate game logic, state management, and interactive UI behavior. This project simulates core Blackjack rules, allowing users to play against a dealer in real time.

## Tech Stack
- HTML
- CSS
- JavaScript (ES6)
- GitHub

## Features
- Interactive card gameplay
- Player vs. dealer logic
- Dynamic score calculation
- Game state tracking (hit, stand, reset)
- Responsive UI for desktop and mobile

## What I Learned
- Managing application state with JavaScript
- Implementing conditional logic and game flow
- Updating the DOM dynamically based on user actions
- Structuring a front-end project for readability and scalability
